# Scientific Calculator in C

## Description

This project is a menu-driven Scientific Calculator developed in C. It performs basic arithmetic operations as well as common mathematical functions using the standard C math library.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Square Root
* Power Calculation
* Sine
* Cosine
* Tangent
* Logarithm (Base 10)

## Technologies Used

* C Programming Language
* Standard C Library
* `math.h` Library

## How to Compile

Use the following command:

```bash
gcc main.c -o calculator -lm
```

## How to Run

### On Windows

```bash
calculator.exe
```

### On Linux/macOS

```bash
./calculator
```
